### Installation


```sh
git clone https://github.com/SaadEssam/hoobank-template-react.git
```

```sh
$ cd hoobank-template-react
```

```sh
$ npm install
```

### Development


```sh
$ npm run dev
```

### Build


```sh
$ npm run build
```
